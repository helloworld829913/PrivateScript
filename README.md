# PrivateScript 私人脚本工具集

这是一个个人使用的实用脚本集合，包含各种自动化工具，用于提高日常工作效率。

## 脚本列表

### extract_audio.bat
从视频文件中提取音频片段的批处理脚本。
- 自动从指定的MP4视频文件中提取音频
- 可以设定开始时间和结束时间来截取特定片段
- 输出为MP3格式音频文件

使用方法：
1. 运行脚本
2. 输入不带扩展名的视频文件名（默认查找.mp4文件）
3. 输入开始时间（格式 HH:MM:SS.xxx）
4. 输入结束时间（格式 HH:MM:SS.xxx）
5. 可选择指定输出文件名（默认为output.mp3）

### ToggleProxy.ps1
PowerShell脚本，用于切换Windows系统代理设置及Git代理设置。
- 开启或关闭Windows系统网络代理
- 同步设置Git的http/https代理
- 显示当前代理状态并提供图形化提示

代理设置：
- IP地址：172.20.10.1
- 端口：1082

### git代理开关.bat
批处理脚本，专门用于快速切换Git代理设置。
- 开启时设置Git全局http/https代理为172.20.10.1:1082
- 关闭时清除Git全局代理设置
- 显示当前Git代理配置状态

### AHK脚本目录
包含多个AutoHotkey脚本，实现各类快捷操作：

- **Handoff4Windows.ahk**: Windows上的接力功能模拟
- **onenoe画图.ahk**: 绘图辅助工具
- **复制粘贴全选替代.ahk**: 替代传统的复制、粘贴、全选操作
- **快速换行.ahk**: 快速换行工具
- **编代码.ahk**: 编程辅助工具
- **随机字符串.ahk**: 生成随机字符串

## 使用说明

根据不同需求选择相应的脚本运行：
- 需要提取视频音频时使用 `extract_audio.bat`
- 需要同时切换系统和Git代理时使用 `ToggleProxy.ps1`
- 只需要切换Git代理时使用 `git代理开关.bat`
- 其他快捷操作可使用AHK脚本目录下的相应工具

## 注意事项

- 部分脚本可能需要管理员权限运行
- 代理相关脚本中的IP地址和端口可根据实际需求修改
- 使用前请确保已安装相关依赖（如FFmpeg等）
